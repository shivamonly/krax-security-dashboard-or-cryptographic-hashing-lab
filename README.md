# KRAX Dashboard

A cinematic, sci-fi inspired web terminal and workspace environment built with React, Vite, and Tailwind CSS. 
Features an immersive video intro sequence with dynamic loading percentage, a retro-styled initialization terminal, and a custom audio toggle player with a waveform animation.

## Development Setup

To run this project locally, follow these steps:

### Prerequisites
- Node.js (v18 or higher recommended)
- npm (comes with Node.js)

### Installation

1. Clone the repository (if you exported it to GitHub) or download the files.
2. Open a terminal and navigate to the project directory.
3. Install the dependencies:
   ```bash
   npm install
   ```

### Running the Development Server

1. Start the Vite development server:
   ```bash
   npm run dev
   ```
2. Open your browser and navigate to the URL provided in the terminal (usually `http://localhost:5173`).

### Building for Production

To create a production-ready build:

```bash
npm run build
```
The compiled files will be output to the `dist` directory.

### Previewing the Production Build

You can preview the built production app locally using:

```bash
npm run preview
```

## Features
- **Cryptographic Hashing Lab**: Real-time message digest generation support with **MD5**, **SHA-256**, and salted **Bcrypt** hashing algorithms.
- **Vulnerability Assessment & Entropy Meter**: Interactive estimation of password strength based on entropy width calculations.
- **Immersive Video Intro**: Dynamic boot sequence loading animation powered by spring-physics counts from `motion/react`.
- **System Terminals**: A retro-themed interactive boot sequence initialization terminal simulating cyber-security network diagnostics.
- **Custom Waveform Audio**: Dynamic wave-form playback animation paired with a persistent toggle for tactical sci-fi ambient soundtracks.
- **Tailwind Sci-Fi Visual Polish**: Uses a fully optimized Material Design token structure layered under monospace typeface and high-contrast terminal indicators.
